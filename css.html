My Journey with CSS

This page will tell you all about my journey with CSS. I like CSS better than HTML, but I think that CSS was also much harder to learn that HTML.
At the beginning, I was so confused on why there were so many different ways to attribute codes, but now that I have a little more experience, it all makes sense.

CSS is fun because this is where you get to personalize the whole site by adding colors, fonts, boxes, borders, and more.
Below are some tricks on how to do my favorite CSS things.

How to get fun fonts:

  • Download fonts off of Google Fonts. You just have to paste in a part of it in HTML and the other part in CSS

  • Google color captchas that you want! For this site, I wanted beige colors, so I literally typed in beige colors>in Google and looked through the images for something I liked.
